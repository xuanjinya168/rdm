# Changelog

本项目的重要变更记录在此文件中。

## 0.2.2（2026-06-25）

### 新增

- 浏览器扩展（实验性）：MV3 扩展拦截浏览器下载或通过右键菜单把链接交给 RDM，自动跳过网页/小文件资源。拦截后桌面端会弹出与「新建下载」相同的确认框（文件名/保存位置/连接数），用户确认后才开始下载，体验对齐 IDM。
- 桌面端本地 HTTP 桥：Tauri setup hook 中起仅监听 127.0.0.1:43721 的 axum 服务，暴露 `GET /health` 与 `POST /downloads`（返回 202 Accepted），通过 `rdm://external-download` 事件把请求交给前端确认框，作为浏览器扩展与桌面端之间的协作通道。

## 0.2.1（2026-06-25）

### 改进

- 完成桌面界面与项目文档的中文本地化补齐。
- Release 校验和仅匹配当前 tag 版本的安装包，避免历史本地产物混入。

### 修复

- 自动化 release EXE 验收在重启后等待设置加载完成，避免主题持久化断言误报。
- 应用 Rust 格式化，确保发布前 `cargo fmt --check` 通过。

## 0.2.0（2026-06-18）

### 新增

- 支持解析 X / Twitter、Instagram 和 Threads 帖子中的图片与视频。
- 下载和媒体解析支持 HTTP、HTTPS、SOCKS5 代理及可选认证。
- 增加亮色、暗色主题和语义化颜色系统。
- 增加 Windows 发布前冒烟测试清单及本地 Range/慢速下载测试服务器。
- 增加直接驱动 release EXE 的 20 项自动化验收，以及 MSI/NSIS 安装包验收，并接入 Windows CI 与 Release。
- Release 自动生成并上传 MSI、NSIS 安装包的 `SHA256SUMS.txt`。

### 改进

- 强化分段响应校验、文件发布、SHA-256 校验和桌面退出流程。
- SHA-256 计算移至阻塞线程池，避免阻塞异步下载任务。
- 前端关键逻辑测试由 2 个扩展到 10 个，覆盖下载表单、设置、代理和任务操作。
- 更新正式应用图标，改善亮色主题对比度和任务界面状态表达。

### 修复

- 活动下载任务不再能通过右键菜单绕过删除限制。
- 修复首次通过 URL 参数启动时，前端尚未监听事件而丢失 URL 的竞态。
- 移除尚未实现的浏览器扩展和网页嗅探入口，避免展示不可用功能。

## 0.1.0（2026-06-13）

- 首个 Rust + Tauri 2 + Svelte 5 重写版本。
- 支持 HTTP/HTTPS 多连接下载、暂停续传、队列、限速、持久化和 SHA-256 校验。
